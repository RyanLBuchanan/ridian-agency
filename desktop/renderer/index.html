        Quick tips
        
          
        
        
        
        
          
          
            
        1. 
              Pick a workflow.
              Business Workflow or Social Media Production from the sidebar.
            
          2. 
            
        2. 
              Run it.
              Type a task or click one of the suggested prompts, then press
              Run workflow (or Ctrl + Enter in the textarea).
            
          3. 
            
        3. 
              Review outputs.
              Once the run completes, the sidebar lists each output —
              click any tab to view that section.
            
          4. 
            
        4. 
              Approve before sending.
              Exports, Google Drive upload, and email send never auto-fire.
              You click the button each time.
            
          5. 
            
        5. 
              Outputs stay local.
              Every run is saved to outputs/ on your machine.
              Click Open artifact folder on the Actions tab to find it.
            
          6. 
          
        
          
        
            Hover a Recent run to ★ pin or × hide it.
            Click the ← arrow at the top to return to the workflow picker.

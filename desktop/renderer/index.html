          1. 
              Pick a workflow.
              Business Workflow or Social Media Production from the sidebar.
            
            2. 
            
          2. 
              Run it.

              Run workflow (or Ctrl + Enter in the textarea).
            
            3. 
            
          3. 
              Review outputs.

              click any tab to view that section.

              You click the button each time.

          5.

              Click Open output folder on the Actions tab to find it.
